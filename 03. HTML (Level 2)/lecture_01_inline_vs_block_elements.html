Inline Vs Block Elements

Block Elements

  • Takes up the full-width available (Whole Block)
  • Start form new line

Inline Elements

  • Takes up only necessary width
  • Don't start form new line

Am I Inline or Block Element?

  • Heading Element ⇒ Block Element

    My name is Roshan Kumar

    I am 23 Years Old.

  • Paragraph Element ⇒ Block Element

    I love to Solving Problems.

    I love to Listening Music.

  • Anchor Tag ⇒ Inline Element
    My LinkedIn Profile   My GitHub Profile
  • Image Element ⇒ Inline Element
