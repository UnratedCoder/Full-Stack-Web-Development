This is an example of Table.

Syntax of table

<table>

<caption> Table Caption </caption>

<tr>

<th>header 1 </th>

<th>header 2 </th>

</tr>

<tr>

<td> data 1 </td>

<td> data 2 </td>

</tr>

</table>

Data Representation

<caption> ⇒ used to display table name

<th> ⇒ used to display table header

<tr> ⇒ used to display table row

<td> ⇒ used to display table data
