Border

It is used to set an element's border

  • border-width

  • border-style

  • border-color

div {

border-width: 5px;

border-style: solid;

border-color: red;

}

Border shorthand

width | style | color

div {

border: 5px solid blue;

}

Border Sides

To control an individual side of the box

  • border-left

  • border-right

  • border-top

  • border-bottom
